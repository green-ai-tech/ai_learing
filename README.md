## ✨ 项目简介
这是一个**持续更新的 AI 学习仓库**，记录我从基础数学、机器学习、深度学习，到大模型应用的完整学习轨迹。

没有浮躁的堆砌，只有踏实的前行：
- 可运行的代码实现
- 清晰易懂的笔记总结
- 从原理到实践的完整思考
- 一个普通人走进 AI 世界的真实旅程

## 📁 包含内容
- 机器学习基础算法（线性回归、决策树、聚类等）
- 深度学习入门（神经网络、CNN、RNN、Transformer）
- 大模型原理与轻量实践
- Python / PyTorch / 工具链使用笔记
- 小项目、Demo 与实验复盘

## 🎯 设计理念
- **清晰**：用简单的话解释复杂的知识
- **可复现**：代码可直接运行，注释完整
- **持续迭代**：随学习进度不断完善
- **开放分享**：愿这份记录，能照亮同路人的一小段路
