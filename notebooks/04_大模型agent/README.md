# 04_大模型 Agent (Large Model Agents)

## 📚 学习主题
本模块主要学习**大语言模型 (LLM)** 与 **智能体 (Agent)** 的核心技术，涵盖环境搭建、提示词工程、工具调用 (Tool Use) 以及多智能体协作系统，是通往现代 AI 应用开发的必经之路。

## 🎯 核心学习路径

### 第一阶段：环境与大模型基础
- **内容**: 开发环境配置、API Key 获取、基础模型调用。
- **重点**:
    - **环境搭建**: 安装 `openai`, `langchain`, `transformers` 等核心库 (01_环境安装)。
    - **模型调用**: 学习如何通过 API 或本地加载的方式与大模型交互。
    - **理解模型**: 掌握 LLM 的基本工作原理、能力边界与局限性 (03_理解模型)。

### 第二阶段：提示词工程 (Prompt Engineering)
- **内容**: 如何写出高效的提示词，引导模型输出期望结果。
- **重点**:
    - **基础技巧**: Zero-shot, Few-shot, Chain-of-Thought (CoT) 提示。
    - **模板化**: 使用模板动态生成提示词，提高代码复用性 (03_提示词模版)。
    - **结构化输出**: 强制模型输出 JSON、XML 等结构化格式，便于程序处理。

### 第三阶段：智能体与工具调用 (Agent & Tools)
- **内容**: 让大模型不仅仅是"聊天机器人"，而是能执行任务的"智能体"。
- **重点**:
    - **智能体基础**: 理解 Agent 的核心组件（规划、记忆、工具使用）(02_智能体)。
    - **消息机制**: 掌握多轮对话中的上下文管理与 Message 传递 (04_消息)。
    - **工具调用**: 教会模型调用计算器、搜索引擎、代码执行器等外部工具 (04_tool)。
    - **综合实战**: 将工具与智能体结合，构建能自主完成复杂任务的系统 (05_工具与智能体)。

## 🚀 快速开始

### 1. 环境准备
确保已安装大模型相关的 Python 库。具体依赖请参考 `01_环境安装.ipynb`。

pip install openai langchain langchain-community transformers

- **进阶**: 学习 **`03_提示词模版.ipynb`** 和 **`04_tool工具的使用.ipynb`**，掌握 Prompt 和 Tool Calling 的核心技术。
- **实战**: 完成 **`05理解工具与智能体.ipynb`**，亲手构建一个能自主使用工具的智能体。

## 📊 关键概念

| 概念 | 解释 | 对应文件 |
|------|------|----------|
| **LLM (大语言模型)** | 基于 Transformer 架构，在海量文本上预训练的语言模型 | `03_理解模型` |
| **Prompt Engineering** | 通过精心设计提示词，引导模型生成高质量输出 | `03_提示词模版` |
| **Agent (智能体)** | 具备规划、记忆、工具使用能力的 AI 系统 | `02_智能体`, `05_...` |
| **Tool Calling** | 让模型在生成文本的同时，输出调用外部工具的指令 | `04_tool...` |
| **Message (消息)** | 多轮对话中的基本单位，包含角色 (user/assistant/system) 和内容 | `04_消息` |

## 💡 学习建议

1. **不要只当聊天**: 大模型真正的威力在于作为"推理引擎"驱动外部工具。重点学习 **Tool Calling** 和 **Agent 架构**。
2. **提示词是核心技能**: 同样的模型，不同的提示词效果天差地别。多练习如何写出清晰、具体、结构化的 Prompt。
3. **理解上下文窗口**: 了解模型的 Token 限制和上下文管理机制，学会如何在多轮对话中有效利用有限的窗口。
4. **关注安全性**: 当模型能执行代码或调用 API 时，必须考虑安全防护（如 Prompt 注入攻击、权限控制）。
5. **动手实战**: Agent 系统非常依赖实践经验。一定要亲手跑通 `05_理解工具与智能体.ipynb`，并尝试添加自己的工具。
